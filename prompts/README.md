# Промпты для LLM анализа

Эта папка содержит шаблоны промптов для анализа изменений в документах через LLM.

## Файлы

- **system_prompt.txt** - Системный промпт, определяющий роль и стиль ответов LLM
- **user_prompt_template.txt** - Шаблон пользовательского промпта с плейсхолдерами для текстов

## Формат шаблонов

### system_prompt.txt
Содержит инструкции для LLM о том, как анализировать изменения и в каком стиле отвечать.

### user_prompt_template.txt
Содержит шаблон с плейсхолдерами:
- `{old_text}` - текст из старого документа
- `{new_text}` - текст из нового документа
- `{context_section}` - дополнительный контекст (путь, страница и т.д.)

## Изменение промптов

Вы можете редактировать эти файлы для изменения стиля и формата ответов LLM.

После изменения файлов перезапустите программу для применения изменений.
