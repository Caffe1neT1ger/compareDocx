# Результаты сравнения

Эта папка предназначена для сохранения результатов сравнения документов в формате Excel.

Файлы результатов будут автоматически создаваться при запуске программы сравнения.

## Пример использования

```bash
python main.py documents/test_document_1.docx documents/test_document_2.docx results/comparison_1_2.xlsx
```
